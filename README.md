# Yeastem
The Yeastem Tool

# About
Yeastem is a game engine developed by Imemyselfitb. 
Written in C++, it was created for support for things that you wouldn't regularly find in a game (in general of course), like ***click-through windows***, or a way to *get data from other applications*, and a *media editor*. Note: These features are yet to be added.

# Instructions
To download the Git Repo, Run `git clone --recursive https://github.com/Imemyselfitb/Yeastem.git`.
